# Kabrios marketing site

Simple static site for `kabrios.com`.

## Files

- `index.html` — landing page
- `styles.css` — styling
- `script.js` — minor interaction

## Deploy options

### GitHub Pages
1. Create a GitHub repo.
2. Upload these files to the repo root.
3. In GitHub, open **Settings → Pages**.
4. Set the source to deploy from the main branch root.
5. For a custom domain, add `kabrios.com` in Pages settings and configure your DNS records at your registrar.

### Other hosts
This site also works on Netlify, Vercel, Cloudflare Pages, or any static host.

## Customize first
- Replace the form action with your CRM or email service.
- Update copy to match your exact product roadmap.
- Add privacy policy, terms, and contact details before launch.
